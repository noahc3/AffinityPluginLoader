# Affinity Plugin Loader

A managed code plugin loader & injector hook for Affinity v3. Load custom code into Affinity and apply runtime patches using the [Harmony](https://github.com/pardeike/Harmony) library — no more patching DLL files on disk.

Supports Windows and Linux (Wine).
